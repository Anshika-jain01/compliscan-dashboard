# CompliScan Dashboard 🛡️

A simple, user-friendly frontend dashboard built to monitor Legal Metrology compliance on e-commerce platforms like Amazon, Flipkart, Myntra, and others.

I built this as a prototype to show regulators how they could check product listings online and ensure sellers are following government rules (like showing correct MRP, net quantity, and manufacturer details).

---

## 🎯 What Does This Dashboard Do?

This dashboard is designed to be the "Control Desk" for a compliance officer. It features four main sections:

1. **Dashboard Overview:** A summary page showing how many products were scanned, total violations found, and the overall compliance score of platforms.
2. **Scanner:** A live interface to start new scanning jobs. Here, a user can pick platforms and categories to monitor in real-time.
3. **Analytics:** Detailed charts, geographic heatmaps, and seller leaderboards to understand where most violations are happening.
4. **Archive:** A storage section to view old records, search for specific products, and download reports for future action.

---

## 🛠️ How It Works in Real Life

Imagine a traffic policeman holding a speed gun to catch speeding cars. 
- The **Device** (scanner) is the speed gun taking pictures of product labels.
- The **Cloud** processes the picture and checks if the label breaks any rules.
- This **Dashboard** is the traffic judge's screen where they review the evidence and take action against non-compliant sellers.

## ⚙️ How It Was Built

This project is fully designed from scratch using Core Web Technologies. No complex backend or heavy frameworks were used—just pure, clean code designed for speed and simplicity.

- **HTML** for creating the layout and structure of the pages.
- **CSS** for making it look modern, responsive, and visually appealing.
- **JavaScript** for adding interactivity, like switching between tabs and showing pop-up messages.
